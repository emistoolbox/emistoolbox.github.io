Modules

The EMIS Toolbox consists of these modules. All modules work with the same dataset.

Reporting Module

Display of bar graphs, data tables and PDF reports. The user can see graphs for any government level and choose the indicator and style of report to display.

Planning Module

Help administrators decide how to distribute resources to schools. The module uses the database to provide information about where resources are needed. Resources can be allocated manually or with the help of the EMIS Toolbox.

Data Management Module

Display of current data, ability to select a new data set or to import data into the current data set.

Data Validation Module

List of schools or government levels were the change in data between years is unexpectedly high. This module can also be used to actually verify that allocations made in the planning tool result in the expected increase in resources.

GIS Module

Ability to import mapping information and display coloured maps showing indicator levels. Maps are clickable to drill down to more detail.
