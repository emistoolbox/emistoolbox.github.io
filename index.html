EMIS Toolbox

The EMIS Toolbox is a pre-build software that can be used to quickly deploy a customized decentralized EMIS solution.

The EMIS Toolbox provides a modular approach. Each module provides one aspect of the whole EMIS. The design philosophie of the EMIS Toolbox aims to address a wide range of real-life problems encountered in the deployment of an EMIS. The EMIS Toolbox aims to provide the flexibility of a custom build EMIS at the cost and feature advantages of pre-build software. The saved resources both in terms of deployment time and cost) can be used to train users of the system and improve the associated processes.
