Features

A summary of the key features of the EMIS Toolbox.

Highly Configurable

Many aspects of the EMIS Toolbox can be changed. the data model (the structure of the education sector, Schools, District, State, Federal),the data fields (what information can be stored at each level),the data sources (how is data imported from external databases). the indicators (name of indicator, how to calculate the indicator and how we can classify the values of the indicator)the resources (name of resource, cost of resource, which indicators they influence, where are they allocated). the reports (how to select data for the report, values on the X axis, how to group data, what to include in a PDF report). Currently English, German and Sindhi are supported. Translations to other languages can be done locally.

Decentralized Model

The EMIS Toolbox can be deployed in a decentralized way. Isolated instances of the software can work on a common dataset and data flows between different these different installations in two ways. First, roll-out of a full dataset. E. g. central government might want to distribute the latest set of data to all decentralized units. Import of new data. E. g. a decentralized unit can submit collected data back to the central unit. The collected data can be imported and used at local level immediately. The data flow is file based and can use normal file sharing methods (e-mailing, uploading through the Internet, memory sticks or writable CD).

Paper Based Reporting

PDF reports can be created in bulk and then distributed to schools. This provides access to information to lower levels in the education sector that do not have access to IT infrastructure.

Flexible Deployment Options

The EMIS Toolbox can run stand-alone on a single PC. It can also be installed on a server on a LAN, provided a whole office with the EMIS Toolbox. Finally the server could be located at an ISP (Internet Service Provider) and expose the data analysis on the Internet.

Comprehensive Training and Support

Various training modules target different users of the EMIS Toolbox. Basic training on how to read reports and respond to them (aimed at parents and school committees).User training on how to use the EMIS Toolbox. Workshop on data validation strategies using the EMIS Toolbox. Technical training on how to configure the system.
